  # Number Pyramid
```
            1  
           1 1  
           2 1  
         1 2 1 1  
       1 1 1 2 2 1  
       3 1 2 2 1 1  
     x x x x x x x x
```


## Solution
The next line represents what is read aloud from the line above.

1
One 1
Two 1(s)
One 2 and One 1
One 1, One 2 and Two 1(s)
Three 1(s), Two 2(s) and One 1

Therefore the answer is:

One 3, One 1, Two 2(s) and Two 1(s)  
13112221
